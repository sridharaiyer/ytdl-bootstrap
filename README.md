# YouTube MP3 Downloader

### Installation

1. `git clone <this repo>`
1. Create a virtual environment.
1. `pip install -r requirements.txt`
1. `export FLASK_APP=app.py`
1. `flask run`
1. Launch `http://127.0.0.1:5000/` in a browser.
